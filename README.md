Proyecto Feria Aula

Este proyecto consiste en una aplicación Android que permite gestionar datos relacionados con una entidad de la feria Aula.
Utiliza una arquitectura cliente-servidor, donde la aplicación móvil se comunica con un servidor Spring que a su vez se
conecta a una base de datos MySQL para realizar operaciones CRUD (Crear, Leer, Actualizar, Eliminar) sobre los datos.

Contenido del Proyecto
El proyecto está estructurado de la siguiente manera:

/Android: Contiene el código fuente de la aplicación Android.
/Eclipse: Contiene el código fuente del servidor Spring.
/Eclipse: Scripts para la creación de la base de datos MySQL.

Requisitos del Sistema:
Android Studio: Para compilar y ejecutar la aplicación Android.
Java Development Kit (JDK): Necesario para compilar y ejecutar el servidor Spring.
MySQL Server: Para alojar la base de datos.

<Instrucciones de Configuración>

Clonar el Repositorio:
git clone <URL_del_repositorio>

Configurar la Base de Datos:
Ejecutar el script de la base de datos ubicado en /Eclipse para crear la estructura de la base de datos y cargar datos de ejemplo si es necesario.

Configurar el Servidor Spring:
Abrir el proyecto del servidor en tu IDE preferido.
Configurar la conexión a la base de datos en el archivo application.properties.
Compilar y ejecutar el servidor.

Configurar la Aplicación Android:
Abrir el proyecto de la aplicación en Android Studio.
Configurar la URL del servidor.
Compilar y ejecutar la aplicación en un dispositivo Android o emulador.
